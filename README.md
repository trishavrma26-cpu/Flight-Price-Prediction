# Flight Price Prediction using Machine Learning

## Project Overview
This project predicts airline ticket prices using Machine Learning techniques on a dataset containing over 300,000 flight records.

## Dataset Features
- Airline
- Source City
- Destination City
- Departure Time
- Arrival Time
- Number of Stops
- Ticket Class
- Duration
- Days Left
- Price

## Exploratory Data Analysis (EDA)
- Distribution of Flight Prices
- Price Distribution by Ticket Class
- Feature Importance Analysis

## Models Used
1. Linear Regression
2. Random Forest Regressor

## Model Performance

### Linear Regression
- R² Score: 0.9099
- MAE: ₹4500.71
- RMSE: ₹6814.94

### Random Forest Regressor
- R² Score: 0.9848
- MAE: ₹1087.07
- RMSE: ₹2801.09
- MAPE: 7.08%

## Important Features
1. Ticket Class
2. Duration
3. Days Left

## Conclusion
Random Forest Regressor achieved excellent performance with an R² score of 98.48%.
